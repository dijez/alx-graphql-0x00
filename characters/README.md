# Character Pagination Queries

This directory contains GraphQL queries that fetch a paginated list of Rick and Morty characters using the characters(page: Int) field.

Each query retrieves the following fields:
- id
- name
- status
- image

The queries included:
- Page 1
- Page 2
- Page 3
- Page 4

Each query has a matching JSON output file containing the result returned by the API.
